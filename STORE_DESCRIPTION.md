### 详细描述

```
HTTP Status - 开发者必备的HTTP调试工具

🎯 核心功能

• 实时显示 HTTP 状态码
在浏览器地址栏直接显示当前页面的 HTTP 响应状态码（200、301、404、500等），无需打开开发者工具即可快速诊断网页状态。

• 完整的重定向链路追踪
自动追踪并显示页面的完整重定向链路（301/302跳转），清晰展示每一步跳转的状态码和目标地址，帮助开发者快速定位重定向问题。

• 详细的响应头信息
查看完整的 HTTP 响应头信息，包括 Content-Type、Cache-Control、Server 等关键字段，便于调试和优化。

• 服务器 IP 地址显示
显示当前网站的服务器 IP 地址，快速了解服务器部署信息。

• 状态码颜色区分
根据状态码类型自动着色：
  - 绿色：2xx 成功响应
  - 橙色：3xx 重定向
  - 红色：4xx/5xx 错误

### Detailed Description

```
HTTP Status - Essential HTTP Debugging Tool for Developers

🎯 Core Features

• Real-time HTTP Status Display
Display HTTP response status codes (200, 301, 404, 500, etc.) directly in the browser address bar, allowing quick diagnosis without opening developer tools.

• Complete Redirect Chain Tracking
Automatically track and display the complete redirect chain (301/302 redirects), clearly showing the status code and target URL of each jump, helping developers quickly locate redirect issues.

• Detailed Response Headers
View complete HTTP response header information, including key fields like Content-Type, Cache-Control, Server, etc., for easy debugging and optimization.

• Server IP Address Display
Show the server IP address of the current website to quickly understand server deployment information.

• Color-coded Status Codes
Automatic color coding by status type:
  - Green: 2xx successful responses
  - Orange: 3xx redirects
  - Red: 4xx/5xx errors

✨ Product Features

✅ Lightweight Design - Zero external dependencies, no browser performance impact
✅ Privacy Secure - All data processed locally, no user information collected
✅ Open Source - Complete source code available on GitHub for free review
✅ Manifest V3 - Uses Chrome's latest extension standard, safe and reliable
✅ TypeScript Development - Type-safe with guaranteed code quality

👨‍💻 Target Users

• Web Developers - Quick debugging of HTTP status and redirect issues
• Frontend Engineers - Check API requests and resource loading status
• QA Engineers - Verify page responses and error handling
• DevOps Personnel - Monitor server responses and deployment status
• SEO Specialists - Check redirect chains and page status codes

🔒 Privacy Commitment

HTTP Status strictly respects user privacy:
• Does not collect any personal information or browsing data
• All data processed only in local browser
• Does not connect to any external servers
• Does not use any tracking or analytics tools
• Complete privacy policy: https://github.com/maicong/HTTP-Status

🛠️ Technical Implementation

• Manifest V3 - Chrome's latest extension specification
• TypeScript - Type-safe code

📖 How to Use

4. Status codes are automatically color-coded for quick identification
